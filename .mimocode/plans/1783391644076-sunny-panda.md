# 错题集功能实现计划

## 概述

为 StudyFlow AI 新增「错题集」功能，支持：
- 自动收集 AI 分析的疑难点 + 闪卡复习答错记录
- 错题集管理页面（筛选、标记掌握、删除）
- 闪卡复习模式（显示答案、标记对错）
- 一键导出 PDF / Word / Excel（含数据统计）

---

## 1. 安装依赖

```bash
pnpm add jspdf jspdf-autotable docx exceljs
pnpm add -D @types/jspdf @types/exceljs
```

- `jspdf` + `jspdf-autotable`: PDF 生成（服务端）
- `docx`: Word (.docx) 生成
- `exceljs`: Excel (.xlsx) 生成，支持样式和多 Sheet

---

## 2. Prisma Schema 新增

**文件**: `prisma/schema.prisma`

新增 `MistakeItem` 模型：

```prisma
enum MistakeSource {
  ai_difficulty   // AI 分析识别的疑难点
  wrong_answer    // 闪卡复习答错
}

model MistakeItem {
  id            Int           @id @default(autoincrement())
  userId        Int
  documentId    Int?
  flashcardId   Int?
  source        MistakeSource
  content       String        @db.Text       // 题目/疑难点内容
  answer        String?       @db.LongText   // 正确答案
  tags          Json?                         // 标签数组
  correctCount  Int           @default(0)    // 答对次数
  wrongCount    Int           @default(0)    // 答错次数
  mastered      Boolean       @default(false) // 是否已掌握
  createdAt     DateTime      @default(now())
  updatedAt     DateTime      @updatedAt

  user          User          @relation(fields: [userId], references: [id], onDelete: Cascade)
  document      Document?     @relation(fields: [documentId], references: [id], onDelete: SetNull)

  @@index([userId, createdAt])
  @@index([userId, mastered])
  @@index([userId, source])
}
```

在 `User` 模型添加 `mistakes MistakeItem[]`，在 `Document` 模型添加 `mistakes MistakeItem[]`。

运行 `npx prisma migrate dev --name add-mistake-item`。

## 3. 新增 API 路由

### 3.1 错题集 CRUD

| 路由 | 方法 | 说明 |
|------|------|------|
| `/api/mistakes` | GET | 获取错题列表（支持 `?source=`, `?tag=`, `?documentId=`, `?mastered=` 筛选） |
| `/api/mistakes` | POST | 新增错题条目（手动添加或闪卡复习触发） |
| `/api/mistakes/[id]` | PATCH | 更新掌握状态 / 答对答错计数 |
| `/api/mistakes/[id]` | DELETE | 删除错题条目 |
| `/api/mistakes/stats` | GET | 获取统计数据（供 Excel 导出用） |
| `/api/mistakes/import-ai` | POST | 从已有文档的 AI difficulties 批量导入错题 |

### 3.2 导出路由

| 路由 | 方法 | 说明 |
|------|------|------|
| `/api/mistakes/export/pdf` | GET | 生成并返回 PDF 文件流 |
| `/api/mistakes/export/word` | GET | 生成并返回 .docx 文件流 |
| `/api/mistakes/export/excel` | GET | 生成并返回 .xlsx 文件流（含多 Sheet 统计） |

## 4. Excel 统计 Sheet 设计

Excel 文件包含 5 个 Sheet：

| Sheet | 名称 | 内容 |
|-------|------|------|
| 1 | 错题详情 | 完整列表：序号、题目、答案、来源、所属文档、标签、答对次数、答错次数、掌握率、添加时间 |
| 2 | 文档统计 | 按文档分组：文档名、错题数量、平均掌握率 |
| 3 | 标签统计 | 按标签分组：标签名、错题数量、平均掌握率 |
| 4 | 时间趋势 | 按周统计：周起始日、新增错题数、累计错题数 |
| 5 | 掌握评估 | 整体统计：总题数、已掌握数、未掌握数、整体掌握率；按来源分类统计 |

## 8. 关键实现细节

### PDF 生成 (jspdf)
- 使用 `jspdf` 创建文档，`jspdf-autotable` 生成表格
- 设置中文字体支持（嵌入字体或使用 unicode）
- 包含：标题、导出时间、错题列表表格

### Word 生成 (docx)
- 使用 `docx` 库创建 Document → Paragraph + Table
- 分节：标题 → 统计概览 → 错题详情列表
- 每道题包含：题目、答案、标签、来源

### Excel 生成 (exceljs)
- 创建 Workbook，添加 5 个 Worksheet
- 每个 Sheet 设置列宽、表头样式
- 数据格式化：掌握率用百分比，时间用日期格式

### 复习模式 UX
- 进入复习模式后隐藏答案区域
- 「显示答案」按钮点击后展开答案
- 两个操作按钮：「掌握了」(绿色) / 「没掌握」(红色)
- 复习结束后显示本轮统计（总数、正确率）
